Blackjack

Technologies Used

  • HTML5
  • CSS3
  • JavaScript

This is a simple blackjack game that includes all the main features except for splitting. Player faces the dealer who will hit until a hard 17 (card total is 17 with ace values as 1) or greater value is reached. Player actions include hit, stand, double down and insurance betting.


All DOM maniuplation is handled through vanilla JavaScript. Cards are loaded in randomly with an animation. There is only one deck of cards, each card being its own object in an array. After each hand, used cards get popped from card array. Once card array is empty, all cards get pushed back into the orginal array.

restaurantFinder

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Fetch API

restaurantFinder is a small web app that will give you nearby restaurant locations based on the given city. Restaurant details include address, buisness hours, average price for two and an aggregate rating.


All information was provided from the Zomato API. The API uses an api-key type authorization. Each call returns thousands of restaurant responses, so I only showed 20 of them because of the limited amount of calls I got with my account. Each restaurant returned will display in card form and can be clicked to show modal with the specfic information.
